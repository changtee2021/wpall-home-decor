# WP ALL Home & Decor — Brand CI

| Token | Hex | Usage |
|-------|-----|--------|
| Primary Teal | `#1F847E` | Links, header accent, secondary CTA |
| Primary Orange | `#F48F33` | Primary CTA (buy, pay, signup) |
| Off-White | `#F1F1F1` | Section backgrounds |
| Text Dark | `#211F20` | Body text |

## Typography

Commercial fonts from brand board: **DB Heavent** (headings), **FC Friday** / **FC Motorway** (body).

Until `.woff2` files are provided, the app uses **IBM Plex Sans Thai** as fallback (`src/styles.css`).

## Assets

- `public/brand/wpall-logo.svg` — simplified logo mark
- Auth card uses teal/orange zigzag pattern strip (CSS in `AuthCardShell`)
